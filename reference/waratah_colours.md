# waratah_colours

Anchor colours used to create the waratah colour palettes. These can
also be used as stand-alone colours (e.g. waratah_colours\$blue_01
returns the hex code \\#002664\\)

## Usage

``` r
waratah_colours
```

## Format

A list with 47 elements

## Details

Colours are available to view at
https://digitalnsw.github.io/nsw-design-system/docs/content/design/theming.html
