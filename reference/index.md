# Package index

## Plot theming

Main theme and colour palettes.

- [`theme_waratah()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/theme_waratah.md)
  : Theme plots using NSW colours and fonts
- [`pal_waratah()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/pal_waratah.md)
  : Flexible colour palettes
- [`pal_nsw()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/pal_nsw.md)
  [`pal_nsw_manual()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/pal_nsw.md)
  : NSW Design System colour palettes

## Colour utilities

More specialised utilities for customising palettes within the design
system.

- [`nsw_colours`](https://digitalnsw.github.io/nsw-r-visualisations/reference/col_nsw.md)
  [`define_colour_theme()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/col_nsw.md)
  [`col_nsw()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/col_nsw.md)
  : Access NSW palette grids by rows and/or columns
- [`pal_interleave()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/ggplot_palettes.md)
  [`pal_c()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/ggplot_palettes.md)
  [`pal_stretch()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/ggplot_palettes.md)
  [`col_contrasting()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/ggplot_palettes.md)
  : Construct palette variants

## HTML elements

Themes for interactive HTML tables and plots.

- [`tooltip_css()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/tooltip_css.md)

- [`reactable_theme()`](https://digitalnsw.github.io/nsw-r-visualisations/reference/reactable_theme.md)
  : Theme for reactable interactive tables
